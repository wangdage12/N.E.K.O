NapCat 启动说明

如果你不是通过插件直接启动 NapCat，请先按下面 4 步手动完成配置，再点击确认。

  1. 下载 OneBot（推荐 NapCat），打开发布页： https://github.com/NapNeko/NapCatQQ/releases
  2. 启动 NapCat，并打开 NapCat 配置页。
  3. 在左侧工具栏进入“网络配置”。
  4. 添加一个 WS 服务器，点击启用，然后保存。
